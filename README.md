# PulseDesk 监测软件 App

这是一个可直接在浏览器打开的监测软件前端原型 App。

## 目录说明

- `index.html`：主界面入口，包含完整交互与深色视觉方案

## 预览方式

直接用浏览器打开 `index.html` 即可预览。

## 当前版本

- 已包含：首页总览、硬件体检、硬件参数/检测、硬件评测、硬件防护/监控、清理优化、驱动检测/管理
- 已包含：筛选、排序、分页、弹窗、详情抽屉、阈值表单、悬浮监控面板、底部实时状态栏
- 已切换为深色版监控风格

## 后续可继续扩展

- 拆分为多文件前端工程
- 接入真实接口与设备数据
- 封装为桌面端安装包
