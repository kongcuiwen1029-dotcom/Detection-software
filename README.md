# PulseDesk 监测软件 App

这是一个可直接在浏览器打开的监测软件前端原型仓库，当前以静态前端工程方式组织，便于后续继续拆分模块、接入真实数据和封装桌面端。

## 项目结构

- `index.html`：页面入口，负责主界面骨架
- `assets/css/styles.css`：页面样式与深色监控视觉方案
- `assets/js/app.js`：页面交互、图表刷新、筛选、分页、弹窗与表单逻辑
- `.editorconfig`：基础编辑规范
- `.gitignore`：本地忽略规则

## 预览方式

直接用浏览器打开 `index.html` 即可预览。

## 当前功能

- 首页总览、硬件体检、硬件参数 / 检测、硬件评测、硬件防护 / 监控、清理优化、驱动检测 / 管理
- 筛选、查询、排序、分页、弹窗、详情抽屉、通知提示
- 阈值配置表单、悬浮监控面板、底部实时状态栏
- 深色版监控风格界面

## 建议的后续开发方向

- 将图表、列表、弹窗拆成更细的组件模块
- 接入真实接口与设备监测数据
- 增加图标、品牌资源与安装包配置
- 按需迁移到 Electron 或 Tauri 桌面端工程
