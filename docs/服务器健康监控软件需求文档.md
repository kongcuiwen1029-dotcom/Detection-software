# 服务器健康监控软件需求文档

## 1. 文档说明

- 文档名称：服务器健康监控软件需求文档
- 版本：V2.1 精简版
- 适用对象：产品、设计、前端、后端、测试、运维
- 目标系统：Windows 10、Windows 11、中标麒麟 Linux 桌面

## 2. 产品目标

本系统定位为服务器健康监控工具，当前阶段只保留核心监控能力，重点解决以下问题：

- 实时查看 CPU 使用率
- 实时查看内存使用率
- 实时查看磁盘空间使用情况
- 预测磁盘预计可用时长
- 定位高占用进程
- 统一处理资源告警
- 配置阈值与白名单

## 3. 本期范围

### 3.1 保留功能

- 总览首页
- 主机监控
- 进程分析
- 磁盘趋势
- 告警中心
- 系统设置

### 3.2 暂不保留

- 报表中心
- 知识库独立页面
- 智能运维独立页面
- 非核心辅助说明区
- 大量解释性文案

## 4. 页面结构

### 4.1 系统形态

- 桌面主窗口
- 悬浮监控球
- 底部实时状态栏
- 弹窗与详情抽屉

### 4.2 主窗口布局

- 顶部状态栏：系统名称、环境、刷新操作
- 左侧导航：总览首页、主机监控、进程分析、磁盘趋势、告警中心、系统设置
- 中央内容区：指标卡片、表格、基础表单

## 5. 核心模块说明

### 5.1 总览首页

首页只保留四个核心数据：

- CPU 使用率
- 内存使用率
- 磁盘空间使用情况
- 预测预计可用时长

补充要求：

- 支持按分钟级刷新
- 支持展示重点主机简表
- 不展示大段辅助说明文案

### 5.2 主机监控

- 展示主机列表
- 展示主机状态
- 展示 CPU、内存、磁盘实时值
- 支持按主机名、IP、操作系统筛选
- 支持刷新数据

### 5.3 进程分析

- 展示高占用进程列表
- 展示所属主机、CPU、内存、持续时长
- 展示白名单状态
- 支持加入白名单

### 5.4 磁盘趋势

- 展示分区列表
- 展示磁盘使用率
- 展示剩余空间
- 展示增长速度
- 展示预计可用时长
- 提供日志整理入口

### 5.5 告警中心

- 展示告警列表
- 展示告警等级
- 展示告警状态
- 支持确认
- 支持忽略
- 支持关闭

### 5.6 系统设置

- 配置 CPU 阈值
- 配置内存阈值
- 配置磁盘阈值
- 配置持续时长阈值
- 配置白名单

## 6. 通用交互要求

### 6.1 查询与筛选

- 支持按主机名查询
- 支持按 IP 查询
- 支持按进程名查询
- 支持按操作系统筛选
- 支持按告警状态筛选

### 6.2 分页与排序

- 列表页支持分页
- 支持设置每页条数
- 支持按 CPU 排序
- 支持按内存排序
- 支持按磁盘使用率排序

### 6.3 弹窗与确认

- 高风险操作需二次确认
- 日志整理需二次确认
- 阈值保存需即时反馈结果

## 7. 表单要求

### 7.1 阈值配置表单

- 字段：CPU 阈值、内存阈值、磁盘阈值、持续时长阈值
- 校验：必填、数值型、范围合法
- 提交：保存后即时生效

### 7.2 白名单表单

- 字段：进程名称、路径、主机类型、保护级别、备注
- 校验：必填、路径合法、不可重复
- 提交：保存后立即参与判断

## 8. 权限要求

- 普通运维：查看监控、查看告警、处理低风险告警
- 高级运维：处理一般告警、发起日志整理申请
- 系统管理员：配置阈值、配置白名单、执行高风险确认
- 审计角色：只读查看

## 9. 设计要求

- 页面尽量简洁
- 首页突出四个核心数据
- 辅助文案尽量减少
- 优先保留数据本身，不堆叠解释性内容
